## 사용 기술 스택
- Python3
- Flask
- Socket.IO
- Jinja2
- MariaDB
- SQLAlchemy

## 구현 기능
1. 재학생 로그인
2. 챗봇 채팅 기능
    1. 유저 - AI 간 채팅 기능
    2. 퀵버튼 기능
3. 스마트 비서 기능
    1. 사용자 일정 저장 및 확인
    2. 교내 최신 공지사항 확인
